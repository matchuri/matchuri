# 프론트엔드 문서 인덱스

프론트엔드 문서는 Next.js 애플리케이션 구현 기준과 API 연동 진입점을 정리합니다.

## 현재 기준

- 구현 가이드: `docs/frontend/guide.md`
- API 계약: `docs/api/index.md`
- 실시간 이벤트 연동: `docs/api/realtime-frontend-guide.md`
- 디자인 시스템 참고 자료: `docs/frontend/references/design-system-reference-llms.txt`

## 주요 문서

- [프론트엔드 가이드](./guide.md)
- [실시간 이벤트 프론트 연동 가이드](../api/realtime-frontend-guide.md)

## 작업 기준

- API 연동 작업은 관련 `docs/api/*.md`를 함께 확인합니다.
- Next.js 관련 코드를 작성하기 전 현재 설치된 `node_modules/next/dist/docs/` 문서를 확인합니다.
- 프론트엔드 개발 기준을 남겨야 할 때는 `docs/frontend/guide.md`를 갱신합니다.
