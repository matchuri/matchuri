# 제품 문서 인덱스

제품 문서는 Matchuri가 어떤 문제를 해결하고, 무엇을 MVP 범위로 보는지 정리합니다.

## 현재 기준

- 제품 감각과 불변 조건: `docs/product/product-sense.md`
- 제품 명세: `docs/product/specs/index.md`
- 도메인 용어: `docs/decisions/domain-language.md`

## 주요 문서

- [제품 감각](./product-sense.md)
- [제품 명세 인덱스](./specs/index.md)
- [그룹 점심 메뉴 결정 명세](./specs/group-lunch-recommendation.md)
- [신규 사용자 온보딩 명세](./specs/new-user-onboarding.md)

## 유지보수 기준

- 제품 판단이 API, 데이터 모델, 운영 기준에 영향을 주면 관련 문서를 함께 갱신합니다.
- 미확정 내용은 `가정(Assumption):`으로 표시합니다.
- 장소/가게 정보는 메뉴 결정 이후의 보조 레이어로 봅니다.
